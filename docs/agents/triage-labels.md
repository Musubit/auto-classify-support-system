# Triage Labels

五个标准 triage 角色与本地 markdown issue 中使用的状态字符串映射。

| 标签 (skills)     | 本地状态              | 含义                       |
| ----------------- | --------------------- | -------------------------- |
| `needs-triage`    | `needs-triage`        | 待维护者评估               |
| `needs-info`      | `needs-info`          | 等待提交者补充信息         |
| `ready-for-agent` | `ready-for-agent`     | 规格完整，可交给 AI Agent  |
| `ready-for-human` | `ready-for-human`     | 需要人工实现               |
| `wontfix`         | `wontfix`             | 不会处理                   |

技能提到 triage 角色时（如 "apply the AFK-ready triage label"），使用此表映射到的本地状态字符串。
